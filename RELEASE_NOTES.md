# Release Notes

## v1.0.1 — Expanded panel hierarchy

- 参考用户提供的布局重新设计展开面板
- 左侧突出本周剩余额度，右侧显示距重置倒计时
- 重置日期、当前套餐和重置机会改为大号粗体摘要
- 展开与折叠状态形成更明显的视觉差异

## v1.0.0 — Initial public release

- Windows 原生 Codex 每周额度悬浮胶囊
- 点击展开详情、点击任意位置回缩、拖动定位
- 按额度比例填充与绿/橙/红状态色
- 平滑逐像素 Alpha 圆角和高 DPI 支持
- 读取重置时间、套餐和重置机会
- 每 60 秒自动刷新
- 系统托盘、窗口找回、立即刷新、开机启动和版本显示
- 自动发现 Codex Desktop / Codex CLI
- 明确的加载、未登录、缺失、超时和协议错误状态
- 无遥测，不保存登录凭证
